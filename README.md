# 🎙️ AI Meeting Assistant

Automatically transcribe meeting recordings and generate summaries, action items, and decisions using AI.

## Features

- 📤 Upload audio/video files (MP3, WAV, M4A, MP4)
- 🎙️ Speech-to-text transcription with Whisper
- 👥 Speaker diarization (identifies who said what)
- 🤖 AI-powered summaries with Gemini API
- ✅ Extract action items and decisions
- 📊 View meeting history
- 🔗 Zoom integration (coming soon)

## Tech Stack

- **Backend**: FastAPI, Whisper, Gemini API, SQLite
- **Frontend**: React, Vite, Axios
- **DevOps**: Docker, Render
